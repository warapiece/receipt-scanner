# レシートスキャナー

スマホでレシートを撮影し、OCRで読み取った内容をGoogleスプレッドシートに自動記録するWebアプリです。

**APIキー不要・完全無料で動作します。**  
OCRにはGoogleドライブの無料OCR機能を使用しています。

## 機能

- 📷 カメラで直接撮影 / ギャラリーから選択
- 🆓 Google Drive OCR による無料テキスト読み取り（APIキー不要）
- ✏️ 読み取り内容の確認・修正
- 📊 Googleスプレッドシートへ自動転記（日付・店舗名・合計金額）

---

## セットアップ手順

### 1. Google Apps Script をデプロイ

1. [Google Apps Script](https://script.google.com) を開く
2. 「新しいプロジェクト」を作成
3. `gas/Code.gs` の内容をすべてコピー＆ペーストして保存
4. **Drive API を有効化**（OCRに必要・無料）
   - 左メニュー「サービス」の **＋** をクリック
   - 「Drive API」を選択 → 「追加」
5. **ウェブアプリとしてデプロイ**
   - 「デプロイ」→「新しいデプロイ」
   - 種類: **ウェブアプリ**
   - 次のユーザーとして実行: **自分**
   - アクセスできるユーザー: **全員**
   - 「デプロイ」をクリック
   - （Googleアカウントのアクセス許可を求められたら承認）
6. 表示された **ウェブアプリのURL** をコピー

### 2. アプリの設定

1. アプリを開き、右上の ⚙️ をタップ
2. 以下を入力して「設定を保存」
   - **GAS エンドポイント URL**: 手順1-6でコピーしたURL
   - **スプレッドシート URL**: 転記先のGoogleスプレッドシートのURL

---

## 使い方

1. トップ画面の「レシートを撮影する」をタップ
2. レシートをカメラで撮影
3. Google Drive OCRが自動で内容を読み取り（数秒〜10秒）
4. 内容を確認・必要であれば修正
5. 「スプレッドシートに記録」をタップ → 完了！
